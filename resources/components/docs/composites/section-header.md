# Section Header

## Purpose

Titles a feed section and provides a single secondary action such as `See all`.

## Anatomy

- Title
- Optional subtitle or supporting copy
- Secondary action link or button

## Behavior

- Primary action: none on the title itself.
- Secondary action: opens the full list or collection view.
- Fallback: if there is no destination, omit the secondary action.
- Feedback: pressed state for the action.
- Tracking: `section_impression`, `section_action_click`.
